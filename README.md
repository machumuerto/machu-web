# Homer's World

A premium animated experience featuring everyone's favorite yellow dad, Homer Simpson.

## Features

- **Interactive Animations**: Feed Homer donuts and watch him react.
- **Dynamic Interactions**: Try to wake him up (if you dare).
- **Smooth Design**: Modern, responsive layout with scroll reveal animations.

## Tech Stack

- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.

## Deployment

Ready for deployment on Vercel.
